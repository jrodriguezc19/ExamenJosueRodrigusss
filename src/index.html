clash royale

En las batallas, los dos adversarios disponen de sus respectivas barajas en rotaciones de cuatro. Según utilices las cartas irán apareciendo otras nuevas para que las sigas combinando a tu criterio. Puedes elegir cartas de ataque, defensa o apoyo dependiendo de las acciones del rival y de tu estrategia. Las cartas consumen elixir para poder ser utilizadas en la batalla. Cada jugador tiene una barra de diez unidades de este elixir que se va recargando progresivamente. Tendrás que estar muy pendiente de cuánto gasta cada carta para hacer las mejores combinaciones y perder el menor tiempo posible.
